# beauty-room


**Описание:**  
„Beauty Room“ е уебсайт, който предоставя информация за правилното гримиране и съчетаване на цветове 
за постигане на елегантни визии. Проектът цели да комбинира полезни съвети с визуални примери
и AI съдържание за по-добро потребителско изживяване.

### Технологии
 - Bootstrap Framework - използван е готов шаблон като основа на проекта, но страниците 
   **Categories** и **Tutorials** са изцяло направени от мен, заедно със CSS стиловете.
 - JavaScript, HTML, CSS - custom логика за Section Observer и за изчисленията за Scrollbar секцията 
   в Color Theory страницата
 - Kapwing, Peppit - за генериране на AI видеа

### Страници
   - **Home** - овна страница представяща уебсайта
   - **Color Theory** - теория за съчетаването на цветовете по най-добрия начин
   - **Tutorials** - галерия с видеа генерирани чрез AI. За трите видеа са използвани 3 различни подхода
     1. _Smokey Eye Video_ - генерирано чрез скрипт, описващ желаните кадри/графики и текст
     2. _Ombré Lips Video_ - генерирано чрез теккстов документ и изззбран аватар
     3. _Winter Glam Video_ - предварително качена снимка и текст, които генерират звук, кадри и субтитри
   - **Categories**
     - **Eye Makeup** - насоки за слагане на сенки, представяне на нови тенденции
     - **Lip Art** - начини за подчертаване на устните, за допълване на визията
     - **Seasonal Trends** - препоръки за ползването на цветови палитри спрямо сезоните

### Добри практики
   - Разделяне на файлове по тип (снимки, видеа, CSS, JS)
   - Responsive дизайн
   - Семантични HTML тагове за по-добра структура и достъпност
